# geotoolbox
Geocaching and Codes Toolbox
Describes "txt" class taking text capable of:
* setting and getting hexadecimal
* setting and getting binary
* setting and getting list of decimal
* XOR
* reversing bitwise
* flipping bitwise

implemented encryption/decryption
* Cesar + ROT13
* RC4

Test text:
* Lorem Ipsum
* English translation
